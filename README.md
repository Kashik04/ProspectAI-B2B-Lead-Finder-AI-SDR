# ProspectAI – B2B Lead Finder & AI SDR

A production-ready AI SaaS web application that 
helps sales teams discover B2B leads, enrich 
contact information, and automatically generate 
personalized outreach messages.

## 🚀 Live Demo
https://prospect-ai-web-app-547768.withmattr.app

## ✨ Features

- 🔍 **AI Lead Finder** — Discover decision makers 
  using intelligent industry, size and location filters
- ✨ **AI Enrichment** — Instantly enrich leads with 
  company insights, tech stack and pain points
- 📧 **AI Outreach Generator** — Generate personalized 
  cold emails, LinkedIn messages and follow-ups
- 📊 **Analytics Dashboard** — Track pipeline 
  performance with real-time charts
- 🎯 **Sales Pipeline** — Kanban board to manage 
  leads from New to Closed
- 🌙 **Dark/Light Mode** — Full theme toggle support
- 📤 **CSV Export** — Export all leads instantly

## 🛠️ Tech Stack

- **React** — Frontend framework
- **Tailwind CSS** — Styling
- **Recharts** — Charts and analytics
- **Lucide React** — Icons
- **Vite** — Build tool
